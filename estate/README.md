**사용 기술**

* recoil 라이브러리를 이용한 상태관리
* kakao map api 를 이용한 지도 구현
* 지도 위에 좌표 데이터를 이용한 폴리곤 구현
* express, mysql의 Ajax 통신을 사용한 front와 back의 데이터 교환
* react-google-charts 라이브러리를 사용한 데이터의 차트화
* MUI 라이브러리를 이용한 select 구현
